Sun and Cloud

Julia Behnen

TCSS 491 Winter 2016 HW3

In this game, each circle selects another circle to be its "sun" and another distinct circle to be its "cloud". The circle wants a sunny day, so it tries to keep its sun directly between itself and its cloud.

To play, select the number of circles and the game type in the settings below, then press "Start" at any time to cancel the existing round and start the new one. "Random" means that suns and clouds are assigned randomly, "Circle" means that the sun-and-cloud arrangment of all the participant circles will allow them to settle into a rotating circle pattern. The exact circle-sun-cloud arrangement for the game is available in the console output as a table.
